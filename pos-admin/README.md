# YO 團購 POS — 分店管理後台

## 說明

本目錄包含分店管理後台的 PHP 原始碼，部署於 VPS 的 Port 9000。

## 功能

- **分店總覽**：顯示所有運行中的分店及其狀態
- **新增分店**：填寫分店名稱、代碼、埠號後，自動執行 `add-branch.sh` 腳本建立新分店
- **快速連結**：每間分店提供「開啟分店系統」按鈕，直接跳轉至該分店 POS 系統
- **伺服器狀態**：顯示 VPS 運行狀態

## 部署方式

管理後台在 VPS 上的部署位置：`/var/www/pos-admin/index.php`

Nginx 設定檔位置：`/etc/nginx/sites-available/pos-admin`

## 存取方式

- **網址**：`http://<VPS_IP>:9000`
- **密碼**：請向系統管理員取得（不在此公開）

## 安全注意事項

- 管理後台使用密碼保護，請定期更換密碼
- 密碼儲存在 `/var/www/pos-admin/index.php` 的 `ADMIN_PASSWORD` 常數中
- 建議設定防火牆，限制 Port 9000 只允許特定 IP 存取

## 新增分店流程

1. 登入管理後台
2. 填寫「分店名稱（中文）」、「分店代碼（英文小寫）」、「埠號」
3. 點選「建立分店」按鈕
4. 等待約 2-3 分鐘，系統自動執行 `add-branch.sh` 腳本
5. 重新整理頁面確認新分店已出現
