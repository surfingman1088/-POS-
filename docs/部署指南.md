# YO 團購 POS 系統 - Oracle Cloud 多分店部署指南

這份指南將協助您在 Oracle Cloud VPS 上，透過一鍵腳本自動部署 6 間分店的獨立 POS 系統。

## 系統架構

本部署方案採用**多租戶獨立架構**，在一台 VPS 上為每間分店建立獨立的：
- 資料庫（pos_bade, pos_sanxia 等）
- 網站目錄（/var/www/pos-bade 等）
- 存取埠號（8001, 8002 等）
- PHP-FPM 處理程序池

這種架構確保各分店的資料完全隔離，營業額、訂單、庫存互不干擾，且其中一間分店系統異常不會影響其他分店。

## 部署步驟

### 1. 登入您的 Oracle Cloud VPS

使用您下載的 `.key` 或 `.pem` 私鑰檔案登入您的 VPS：

```bash
# 如果您使用 Mac 或 Linux：
chmod 400 您的私鑰檔案.key
ssh -i 您的私鑰檔案.key ubuntu@您的VPS_IP

# 如果您使用 Windows (PowerShell)：
ssh -i 您的私鑰檔案.key ubuntu@您的VPS_IP
```

### 2. 下載並執行一鍵部署腳本

登入 VPS 後，依序複製並貼上以下三行指令並按 Enter 執行：

```bash
# 下載部署腳本
curl -O https://raw.githubusercontent.com/surfingman1088/-POS-/main/deploy-vps.sh

# 賦予執行權限
chmod +x deploy-vps.sh

# 以管理者權限執行（過程約需 5-10 分鐘）
sudo bash deploy-vps.sh
```

### 3. 取得分店網址與密碼

部署完成後，畫面上會顯示所有分店的專屬網址和資料庫密碼。這些資訊也會自動儲存在 VPS 的 `/root/yo-pos-deploy-info.txt` 檔案中。

預設的分店網址如下：
- **八德店**：`http://您的VPS_IP:8001`
- **三峽店**：`http://您的VPS_IP:8002`
- **大竹店**：`http://您的VPS_IP:8003`
- **林口店**：`http://您的VPS_IP:8004`
- **藝文店**：`http://您的VPS_IP:8005`
- **菓林店**：`http://您的VPS_IP:8006`

> **重要安全提示**：所有分店的預設管理員帳號為 `admin`，密碼為 `123`。請務必在首次登入後，立即前往「設定 > 密碼」修改密碼！

## 系統管理與擴充

### 如何新增分店？

未來若有新分店開幕，只需在 VPS 上執行新增分店腳本，系統會自動分配下一個埠號並建立獨立環境：

```bash
sudo bash /opt/yo-pos/scripts/add-branch.sh
```
執行後，系統會提示您輸入新分店的 ID（英文）、名稱和地址，確認後即自動完成部署。

### 如何重啟系統？

如果系統卡住或修改了設定，可以執行以下指令重啟服務：

```bash
sudo systemctl restart php8.3-fpm nginx
```

### 如何查看日誌？

各分店的錯誤日誌位於：
```bash
# 查看八德店的日誌
sudo tail -f /var/www/pos-bade/storage/logs/laravel.log
```
