# YO 團購 POS 系統 - Windows 本機安裝指南

本指南說明如何在 Windows 電腦上安裝並測試 POS 系統。

---

## 系統需求

| 軟體 | 版本 | 下載連結 |
|------|------|---------|
| Git | 最新版 | https://git-scm.com/download/win |
| Laragon | 最新版 | https://laragon.org/download/ |
| PHP | 8.2 以上（Laragon 內建） | - |
| Composer | 最新版 | https://getcomposer.org/download/ |

## 安裝步驟

### 第一步：安裝 Laragon

1. 下載並安裝 [Laragon](https://laragon.org/download/)（建議選 Full 版本）
2. 安裝完成後開啟 Laragon，點選「**Start All**」啟動服務

### 第二步：下載 POS 系統代碼

開啟 Laragon 終端機（右鍵 Laragon 圖示 → Terminal），執行：

```bash
cd C:/laragon/www
git clone https://github.com/surfingman1088/-POS-.git yo-pos
cd yo-pos
```

### 第三步：安裝依賴

```bash
composer install
npm install
npm run build
```

### 第四步：設定環境變數

```bash
copy .env.example .env
php artisan key:generate
```

開啟 `.env` 檔案，修改以下設定：

```env
APP_NAME="YO 團購 POS - 測試店"
APP_URL=http://yo-pos.test
APP_LOCALE=zh

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=yo_pos_test
DB_USERNAME=root
DB_PASSWORD=
```

### 第五步：建立資料庫

1. 開啟瀏覽器，前往 `http://localhost/phpmyadmin`
2. 點選「新增」，建立資料庫名稱：`yo_pos_test`
3. 字元集選擇：`utf8mb4_unicode_ci`

### 第六步：執行資料庫遷移

```bash
php artisan migrate
php artisan db:seed
php artisan storage:link
```

### 第七步：存取系統

開啟瀏覽器，前往：
```
http://yo-pos.test
```

或（如果虛擬主機未設定）：
```
http://localhost/yo-pos/public
```

**預設帳號：**
- 使用者名稱：`admin`
- 密碼：`123`

## 設定自動虛擬主機（Laragon）

Laragon 支援自動虛擬主機，讓您可以用 `http://yo-pos.test` 存取。

1. 右鍵 Laragon 圖示 → **Preferences**
2. 確認「**Auto Virtual Hosts**」已勾選
3. 重啟 Laragon（Stop All → Start All）
4. 瀏覽器前往 `http://yo-pos.test` 即可

## 常見問題

**Q：`php artisan` 指令找不到？**

請確認 PHP 已加入環境變數 PATH：
1. 搜尋「環境變數」→「編輯系統環境變數」
2. 點選「環境變數」→「系統變數」→「Path」→「編輯」
3. 新增 Laragon 的 PHP 路徑，例如：`C:\laragon\bin\php\php-8.x.x`
4. 重新開啟終端機

**Q：資料庫連線失敗？**

確認 `.env` 中的 `DB_PASSWORD` 是否正確（Laragon 預設 MySQL 密碼為空）。

**Q：頁面顯示 404？**

確認 `php artisan storage:link` 已執行，且 Laragon 的 Document Root 指向 `public` 目錄。

## 升級系統

當 VPS 上的系統有更新時，本機也可以同步：

```bash
cd C:/laragon/www/yo-pos
git pull origin main
composer install
npm install
npm run build
php artisan migrate
php artisan config:cache
```
