## 員工管理功能說明

> **注意**：「員工管理」與「帳戶管理」是兩個不同的功能。
>
> - **員工管理**：管理外送員、店員等工作人員資料（姓名、聯絡電話、狀態）
> - **帳戶管理**：管理可以登入 POS 系統的使用者帳號（帳號、密碼、角色）

### 新增員工

1. 以任意帳號登入分店系統
2. 左側選單 → **Management（管理）** → **Employees（員工）**
3. 點擊右上角「**新增員工**」按鈕
4. 填寫姓名、09 開頭的 11 位手機號碼、在職狀態
5. 點擊「儲存」完成新增

### 封存員工

員工離職時可將其封存，保留歷史訂單記錄。若員工有進行中的訂單，系統將拒絕封存操作。

## 帳戶管理功能說明（僅管理員）

### 建立員工帳號

1. 以**管理員**帳號登入分店系統
2. 左側選單 → **Management（管理）** → **Accounts & Sessions（帳戶管理）**
3. 點擊「**建立帳號（Create account）**」
4. 填寫姓名、帳號、初始密碼，並在「**Role（角色）**」欄位選擇 `Staff — Limited access`
5. 點擊「建立帳號」完成

### 員工首次登入流程

員工帳號建立後，首次登入時系統會強制要求設定自己的密碼，完成後才能正常使用系統。帳戶列表中尚未完成設定的帳號會顯示 **「Password not set」** 紅色標籤。

---

## 商品出庫管理員密碼驗證

當**員工**嘗試修改商品庫存數量時，系統會彈出「管理員授權」視窗，需由管理員輸入密碼授權後才能儲存。管理員直接操作不受此限制。

---

## 商品成本設定

在商品管理頁面新增或編輯商品時，可設定每個商品的**成本（Cost）**欄位，系統將依實際成本計算利潤，取代固定利潤率估算。

## 快速操作指南

### 一、全新安裝（新 VPS）

```bash
curl -O https://raw.githubusercontent.com/surfingman1088/-POS-/main/scripts/install.sh
chmod +x install.sh
bash install.sh
```

### 二、升級系統（從 GitHub 拉取最新代碼）

```bash
bash /opt/yo-pos-source/scripts/upgrade.sh
```

### 三、新增分店

```bash
bash /opt/yo-pos-source/scripts/add-branch.sh <分店ID> <分店名稱> <埠號>

# 範例：新增中壢店（Port 8007）
bash /opt/yo-pos-source/scripts/add-branch.sh zhongli 中壢店 8007
```

### 四、備份資料庫

```bash
bash /opt/yo-pos-source/scripts/backup.sh
```

設定每日自動備份（凌晨 2 點）：

```bash
crontab -e
# 加入以下一行：
0 2 * * * bash /opt/yo-pos-source/scripts/backup.sh >> /root/backup.log 2>&1
```

### 五、查看系統狀態

```bash
bash /opt/yo-pos-source/scripts/status.sh
```

## 常見問題

**Q：如何修改管理員密碼？**
登入後點選左下角帳號名稱 → 帳戶設定 → 修改密碼

**Q：如何新增商品？**
登入後點選左側選單「產品」→ 點擊「新增產品」

**Q：員工看不到某些選單項目？**
折扣設定、帳戶管理、系統日誌為管理員專屬功能，員工帳號不顯示。

**Q：系統無法存取怎麼辦？**
```bash
bash /opt/yo-pos-source/scripts/status.sh
```
